# Музыкальная база данных (PostgreSQL)

## Описание
База данных для музыкального сервиса с поддержкой many-to-many связей между исполнителями, жанрами и альбомами.

## Структура БД

### Основные таблицы:
- **genres** - жанры музыки
- **artists** - исполнители
- **albums** - альбомы
- **tracks** - треки (принадлежат одному альбому)
- **collection** - сборники

### Связующие таблицы (many-to-many):
- **artist_genre** - связь исполнителей и жанров
- **artist_album** - связь исполнителей и альбомов
- **collection_track** - связь сборников и треков
